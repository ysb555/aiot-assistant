# 智能门锁系统——语音识别开锁

用窗帘机的模拟门锁的开关状态，实现语音识别的开锁操作，并额外添加了wifi控制的模块，可以通过wifi发送指定命令进行开锁，还与传感器进行联动。

## 任务设计：

（1）配置wifi模块：确保可以远程客户端进行链接

（2）语音控制：实时识别指令（如“打开窗帘”、“切换风扇档位”）并执行操作。

（3）wifi控制：发送open curtain和close curtain等进行指定操作。

（4）开锁时发出指定提示音，开锁后后延迟几秒后自动关锁。

## 任务实现：

（1）语音识别线程：实时解析指令 → 主线程处理指令 → 执行门锁操作，使用 `sherpa-ncnn-alsa-1`语音引擎实时捕获音频并处理，与指令文字相匹配，判断是否为指定命令，并进行相应操作。

（2）Wi-Fi控制模块：WiFi模块初始化 → TCP长连接 → 指令监听 → 安全执行，**初始化**配置后与用户端建立TCP连接，读取用户发来的命令，解析并匹配。

（3）开锁后播放提示音文件（`play_feedback_sound`函数），主循环中检查开锁，若开锁：延迟五秒（sleep(5);）后，执行关锁操作（curtain_control(param_e3, 0);）。
