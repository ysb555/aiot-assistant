# 智能门锁系统——NFC开锁

用窗帘机的模拟门锁的开关状态，实现NFC的开锁操作，此处尚不支持向NFC卡写入内容并进行指定内容的识别，会在后续任务（Smart_lock_pro）中完成。

## 任务设计：

（1）按开锁键SWC，电机工作，发出开锁提示音。

（2）用按键SWA进入设定密码的模式（密码设置为4位），输入密码，电机工作，发出开锁提示音。（此处的密码开锁时，数码管显示存在问题，正确实现参照Task7-1，或最后的lock_pro的代码实现）

（3）刷NFC，电机工作，发出开锁提示音。

（4）开锁后延迟几秒后自动关锁

## 任务实现：

（1）检测按键的值，如果是SWC，直接打开锁，几秒后自动关锁（对应窗帘机移动到100的位置，延迟几秒后移动到0的位置）；如果是SWA，则进入设置密码的模式，输入四位数字作为新密码；如果是按键的值对应的0～9的数字，默认进入输入密码的模式，输入满四位进行匹配（此处的密码开锁时，数码管显示存在问题，正确实现参照Task7-1），成功则打开锁，失败则关锁。

（2）接入数码管rgb灯的组件进行辅助：为进行任何操作时数码管显示0000，正常输入密码时rgb亮红灯，输入密码时数码管显示当前输入的密码，按开锁键实现的开锁为1111，输入密码的开锁显示2222，识别NFC成功后开锁显示3333，密码错误时显示8888，开锁时rgb亮绿灯，重置密码时rgb亮蓝灯。
